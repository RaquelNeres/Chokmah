FrontEnd

Tailwind

  • 1. Estilo Visual (Texto & Cores)

    text-{cor}-{tom}: Muda a cor da fonte.

    text-{size}: Tamanho (xs, sm, base, lg, xl, 2xl...).

    text-{align}: Direção (left, center, right, justify).

    bg-{cor}-{tom}: Cor de fundo.

    hover:: Efeito ao passar o mouse.

    hover:bg-blue-400

  • 2. Espaçamento & Dimensões

    m- / my- / mx-: Margem (externa).

    p- / py- / px-: Padding (interno).

    w-{size}: Largura (w-full, w-1/2, w-[100px]).

    h-{size}: Altura (h-screen, h-full, h-10).

    rounded-full: Borda totalmente redonda.

    p-4
    rounded-full

  • 3. Layout (Flex & Grid)

    flex: Alinha elementos em 1 eixo.

    gap-{size}: Espaço entre elementos.

    grid grid-cols-{n}: Cria uma grade de N colunas.

    col-span-{n}: Estica o elemento por N colunas.

    01
    02
    03
    col-span-3 (Ocupa tudo)

  • 4. Breakpoints (Telas)

    Prefixo Mínimo Uso Comum
    (Base) 0px Celular (Padrão)
    sm: 640px Celular Deitado
    md: 768px Tablets / iPads
    lg: 1024px Laptops
    xl: 1280px Desktops
    2xl: 1536px Telas Gigantes
